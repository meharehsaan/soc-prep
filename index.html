With 2 years of hands-on experience in detection engineering, threat intelligence, and SIEM operations, the profile is already strong. The gap between current level and international-vendor hiring is not experience — it's visibility, specificity, and a few technical credentials that signal serious depth.

At international vendors (EU, US remote), a Detection Engineer or Threat Intel Analyst L2 with this profile realistically targets $90k–$140k USD depending on location. The ceiling-breakers are published open-source tools with GitHub traction, conference talks, and GREM/GCTI credentials.

03

Skill gaps to close

Priority order
■
STIX/TAXII & MISP critical
Industry-standard CTI exchange formats. Required at every major intel vendor. Build a MISP instance and publish to it from your IOC pipeline. STIX 2.1 bundle generation should be in your GitHub.
■
Cloud security & log sources high
AWS CloudTrail, Azure Sentinel / Entra ID, GCP SCC. All top vendors operate cloud-native. At minimum get AWS Security Specialty or AZ-500 cert. Know what CloudTrail events look like for common attacks.
■
KQL & SPL query proficiency high
Microsoft Sentinel (KQL) and Splunk (SPL) dominate enterprise SOCs. Add query libraries to GitHub. Write detection queries from memory for password spray, lateral movement, C2 beaconing. These come up in first-round interviews.
■
Certifications — priority order medium
GREM (malware) → GCTI (intel) → BTL2 or GDAT (detection) → OSCP (valued at Rapid7/Mandiant for detection roles). Skip CompTIA — it signals entry level at this experience tier.
■
Detection-as-code & CI/CD medium
Docker + GitHub Actions for detection pipelines. Run Sigma rule linting and unit tests on every commit. This shows engineering maturity beyond analyst-level work and is specifically valued at CrowdStrike, Elastic, and SentinelOne.
04

Expert mastery points

What interviewers probe
ATT&CK depth
Know sub-techniques cold. Be able to map a raw log line to a technique live. Build ATT&CK Navigator layers for published actors during prep.
Detection logic
Explain baselining, false-positive tuning, and risk scoring. Know the difference between behavioral and IOC-based detection and when each fails.
Malware RE
Static + dynamic analysis. Able to extract C2 configs, identify packers, write YARA. IDA/Ghidra proficiency beyond surface level. Know FLOSS for obfuscated strings.
Intel lifecycle
Full F3EAD cycle. Diamond Model. Know how to assess reliability, source credibility, and confidence levels. Admiralty Scale applied in practice.
Python automation
Clean, documented, tested Python. Real CLI tools with argparse, proper error handling. No Jupyter-only code for production tools. Published to PyPI is a bonus.
Write-ups
Publish threat analysis blog posts on Medium or your own site. Mandiant and Recorded Future heavily value written intelligence products as portfolio evidence.
